# 🌿 VangVieng Explorer

Tourism web app for Vang Vieng, Laos. Built phase by phase.

## ໂຄງສ້າງ Folder

```
vangvieng-explorer/
├── index.html          ← Home page ✅
├── css/
│   └── style.css       ← Design system ✅
├── js/
│   └── app.js          ← Logic + mock data ✅
├── pages/
│   ├── explore.html    ← Phase 2 (ຍັງບໍ່ທັນ)
│   ├── map.html        ← Phase 3 (ຍັງບໍ່ທັນ)
│   ├── detail.html     ← Phase 3 (ຍັງບໍ່ທັນ)
│   └── ai-chat.html    ← Phase 4 (ຍັງບໍ່ທັນ)
└── assets/             ← ຮູບ, icons (ເພີ່ມທີຫຼັງ)
```

## ວິທີ Run (Phase 1)

1. ດາວໂຫຼດ folder ທັງໝົດ
2. ເປີດ `index.html` ດ້ວຍ browser ໄດ້ທັນທີ
3. ຫຼື install VS Code + Live Server extension ສຳລັບ auto-reload

- Frontend: HTML + CSS + Vanilla JavaScript
- Database: Supabase (Phase 2+)
- AI: Google Gemini API (Phase 4+)
- Hosting: Vercel (Phase 5)
- Cost: ~$15/ປີ (domain only)
